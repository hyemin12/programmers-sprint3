# React와 Typescript를 사용한 BookStore 프로젝트

## 💡사용 기술

- React
- Typescript
- styled-components
- zustand
- tanstack query(react query)
- react-hook-form
- axios
- Dayjs

## 💡실행 방법

1. 프로젝트 클론

```bash
git clone https://github.com/hyemin12/programmers-sprint3.git
```

2. 필요한 패키지 설치

```bash
npm install
```

3. 프로젝트 실행

```bash
npm run start
```
